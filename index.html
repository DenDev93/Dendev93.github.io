Sobre Mí

Denis Manuel López Molina

Ingeniero en Sistemas de Información

Universidad Nacional Autónoma de Nicaragua - UNAN León

Desarrollador Web Full-Stack Jr | Diseñador Gráfico

Soy un desarrollador apasionado por crear soluciones web innovadoras y eficientes. Mi enfoque principal es en la experiencia del usuario (UX) y el diseño de interfaces (UI), asegurando que cada proyecto no solo sea funcional, sino también intuitivo y visualmente atractivo. Siempre estoy aprendiendo nuevas tecnologías y metodologías para mantenerme al día con las últimas tendencias de la industria.

Habilidades

Desarrollo Web & Programación

  • Frontend: HTML5, CSS3 (Sass, Less), JavaScript (ES6+), React.js, Vue.js, Angular
  • Backend: Node.js (Express), Python (Django, Flask), PHP (Laravel), Java (Spring Boot)
  • Frameworks/Librerías: Bootstrap, Tailwind CSS, jQuery
  • Control de Versiones: Git, GitHub

Administración de Bases de Datos

  • Relacionales: MySQL, MariaDB, PostgreSQL, SQL Server, Oracle, PLSql
  • No Relacionales (NoSQL): MongoDB, Firebase Firestore
  • ORM/ODM: Sequelize, Mongoose, SQLAlchemy

Diseño Gráfico

  • Herramientas: Adobe Photoshop, Adobe Illustrator, Figma, Sketch, Entre Otros
  • Conceptos: Diseño UI/UX, Principios de Diseño, Tipografía, Teoría del Color, Identidad de Marca, Imagen Corporativa

Metodologías & Otros

  • Metodologías Ágiles: Scrum, Kanban
  • API RESTful: Diseño y consumo de APIs RESTful
  • Optimización: Optimización de Rendimiento Web
  • Testing: Debugging & Testing (Unitarias, Integración)
